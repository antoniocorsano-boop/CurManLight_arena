# CML-608 — Onboarding Role Selector Contrast Fix

## Causa

`bg-primary-600` non è definito nel Tailwind config. La classe genera sfondo trasparente + `text-white` = testo bianco su sfondo bianco. Colpito anche `bg-primary-100`, `bg-primary-50`, `border-primary-*` in 14 occorrenze.

## File modificati

| File | Modifica |
|------|----------|
| `tailwind.config.js` | Aggiunta definizione colori `primary` (50–950) come alias di indigo |
| `src/features/session/components/SessionModals.tsx` | Stato disabled "Precedente": `text-slate-300` → `text-slate-400` |

## Regola corretta

- `primary` ora è un set di colori valido nel Tailwind config
- Tutte le 14 occorrenze `bg-primary-*` funzionano correttamente
- Stato selezionato: sfondo viola + testo bianco
- Stato non selezionato: sfondo chiaro + testo scuro
- Stato disabled: testo leggibile (`slate-400`)
