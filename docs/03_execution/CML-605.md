## Eventi coperti

- PDF — curricolo (2 handler)

## Eventi esclusi
- Copia negli appunti (clipboard) — non produce file
- Reset memoria — non e un'esportazione
- Backup JSON — gestito da useBackupHandlers
- Genera Programmazione/Relazione/Specifico — producono solo testo, il PDF avviene in SessionModals
- Template AI — flusso complesso, escluso dal perimetro

## Contratto coerenza
- current: firma del contenuto uguale (stessa sorgente, stesso stato)
- modified: firma del contenuto diversa (la sorgente e stata modificata dopo l'esportazione)
- unverifiable: senza firma iniziale, senza sorgente, o sorgente aggregata non confrontabile
- exportedAt: solo informazione visiva, mai usata nel confronto

## Limiti
- La coerenza confronta solo lo stato locale, non il file reale
- I file restano sul dispositivo, non sono conservati nell'app
- Max 5 eventi (le vecchie esportazioni vengono eliminate)
- Nessuna verifica dell'esistenza del file

## Rischi residui
- Docenti con molteplici esportazioni settimanali potrebbero percepire il limite di 5 come insufficiente
- La coerenza confronta lo stato locale sorgente, non il file esportato sul dispositivo
- I casi di tipo 'programmazione' o 'relazione' sono classificati unverifiable perche la sorgente e aggregata
